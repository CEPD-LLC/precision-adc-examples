Detailed Description

Hardware abstraction layer (HAL) implementation.

Warning
This HAL implementation is provided as an example of how to interface to TI SysConfig supported processors (https://www.ti.com/tool/SYSCONFIG). This code will need to be re-implemented to target other processors!
